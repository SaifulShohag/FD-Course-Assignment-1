1. HTML Assignment: Terminology and Definitions

1. HTML & CSS: HTML is a programming language that stands for Hypertext Markup Language This is a relatively simplistic yet powerful programming language that allows web developers and website owners to create the structure of their websites.
CSS, stands for "Cascading Style Sheets" is a style sheet language thatâ€™s focused almost entirely on improving the presentation of HTML elements. While CSS is a static programming language, it can be used to make your website appear visually pleasing and modern.

2. HTML or CSS for putting paragraph of text on a webpage: HTML is used to structure the content on the web page. CSS is used to add style to the content of a web page. Therefore without HTML I wouldn't be able to add any text in any webpage. So, I would use HTML.

3. HTML or CSS for changing font and background color of button: We can't change a button's background color in HTML5 anymore, but we can use "font" tag and it's attributes "face", "size" and "color" to change a button's font. An example is given below:

Source Code: <'button><'font face="arial" size="3" color="blue">Click me<'/font><'/button> (ignore the single quotation marks)

But for styling HTML tags, using CSS is much more neat and reusable. We can also use "style" HTML attribute to write CSS in a button tag and change it's background color and font.

4. HTML Tag: An HTML tag is a piece of markup language used to indicate the beginning and end of an HTML element in an HTML document. As part of an HTML element, HTML tags help web browsers convert HTML documents into web pages.

5. Three parts of an HTML elements: a start tag, some content, and an end tag.

6. Purpose of DOCTYPE declaration: The DOCTYPE declaration is an instruction to the web browser about what version of HTML the page is written in. This ensures that the web page is parsed the same way by different web browsers.

7. HTML Element: After we declare the doctype, we need to provide an "html" element. This is what's known as the root element of the document, meaning that every other element in the document will be a descendant of it

8. Purpose of "head" element: The element is where we put important meta-information about our webpages, and stuff required for our webpages to render correctly in the browser. Inside the "head", we should not use any element that displays content on the webpage.

9. Purpose of "body" element: The final element needed to complete the HTML boilerplate is the "body" element. This is where all the content that will be displayed to users will go - the text, images, lists, links, and so on.

10. Create a paragraph in HTML: With the help of "p" tag which has some default top and bottom margin.

11. Create heading in HTML: There's six heading tag in HTML: h1, h2, h3, h4, h5 and h6. Fonts inside these tags are bold by default and they each has different default font size.

12. Types of heading and their difference: There's six heading tag in HTML: h1, h2, h3, h4, h5 and h6. Fonts inside these tags are bold by default and they each has different default font size.

13. Make text bold and important: In HTML to mark text as both bold and important we should use "strong" tag.

14. Make text italicized to add emphasis: we should use "em" tag.

15. Relationship of HTML elements: Ancestors, Parents, Children, Siblings, Descendants

16. Two elements at the same level of nesting: They are called Siblings.

17. HTML Comments: Comments in HTML start with <!-- and end with --> . Don't forget the exclamation mark at the start of the tag! But you don't need to add it at the end.

18. HTML to create an unordered list: "ul" tag is used to create unordered list.

19. HTML to create an ordered list: "ol" tag is used to create ordered list.

20. HTML tag to create list items forn both unordered and ordered lists: "li" tag is used to create list item for both of these type lists.

21. HTML element to create link: "a" tag (or anchor element), with its "href" attribute, creates a link.

22. HTML Attribute: HTML attributes are special words used inside the opening tag to control the element's behaviour. HTML attributes are a modifier of an HTML element type.

23. Link's destination attribute: The hypertext reference, or "href" attribute is used to specify a target or destination for a link.

24. Difference between an absolute and relative link: An absolute link provides the complete location information of your site, while a relative link only contains the location following your domain.

25. Element to display an image: "img" tag is used to display an image in HTML.

26. Must have two attributes an image: The "img" element has two required attributes. "src" : The source location (URL) of the image file. "alt" : The alternate text, is used to describe the image for someone who cannot see it because they are either using a screen reader or the image src is missing.

27. Access a parent directory of a file: we can navigate to the parent directory of a file by "../".

28. Four main image formats that can be used for images on the web: JPG, PNG, GIF and SVG.
